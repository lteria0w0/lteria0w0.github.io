html介绍

HTML

web 服务器，使得用户任意访问自己所需的网站，浏览所需的信息，成为了可能，并且 web 服务器只要启动，可以不知疲倦的工作，无需考虑空档期。因此，其逐渐取代广播明星的地位。web 服务器页面主要由**html 语言**书写。
html 语言可以在多种平台部署，例如：windows，mac

详情

html 语言是一个**标签语言**，由各类标签组成

每一个标签都有其**属性**，决定标签的具体效果，如决定类型的 type，决定引用地址的 href 属性。
绝对路径：/开头（从根目录开始）;相对路径：文件名开头（从当前文件夹开始）

直接写代码来构建 html 页面，会因为语法而限制页面的**布局等，因此先构建一个粗略的草图**是很有必要的（最基本骨架）

如果想要在段落中添加加粗字体，则可以使用元素嵌套[^1]。

图像可以使得页面更加美观，因此 html 对图像处理也有着相应的标签

创建完成后，就可以发布页面[^2]了，不过首先需要一个域名[^3]来方便用户访问
如果图片过大，可以使用缩略图的形式（连接+小图）

对于页面整体的排版，通常使用盒模型[^5]（div，span）

如果要向服务器**发起请求，可以用超链接或者表单**。

可以通过访问W3C[^4]来验证网页是否符合规范

[^1]元素嵌套：一个标签嵌套在另一个标签内部
[^2]发布页面可以通过托管一家公司
[^3]域名可以通过自己申请或者云平台购买
[^4]W3C 网址：validator.w3.org
[^5]由内外边距，边框，内容组成
