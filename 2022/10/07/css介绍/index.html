CSS

如果页面仅仅是依靠html，那么界面将会十分单一，缺少足够的艺术效果，因此引入了CSS对其进行页面的美化。下面是三种引入css样式的方法。css通过选择器[^1]获取需要设置的标签（对状态的选择通常用伪类选择器）。选择器具有优先级，通过000（id，类/伪类，元素名）方式层叠进行。
选择器具有继承特性，当同时选定一个标签时，满足子类覆盖的特性

下面是常用的一些样式

在思考布局时，元素的上下层关系必须要考虑，此时可以使用浮动元素[^2]（如果发生重叠现象，可以用clear样式解决）
对于界面大小的调整，可以选择：流体，冻结，凝胶

[^1]三种选择器：类选择器.，标签选择器（默认），id选择器#
[^2]浮动元素：漂浮在上方，内联元素会考虑其位置
